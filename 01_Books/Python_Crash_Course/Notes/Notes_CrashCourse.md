#Notes from the book

##Chapter 1- Getting Started 
- Essentailly just review on getting set up in python and using VS code
- Learned how to navigate the terminal to run a program. 
- Use "pwd" to see where I am, "cd" to change working directory, ls to see what files are in the folder we are currently in, and then use python3 file_name in the terminal to run the program. 

***One thing to Remember: Need to use "" if path name has spaces in it because python sees that as a break, so "" go around that."***

##Chapter 02--Variables and Types of Data
- Not much new information, essentially just learned about types of data
- e.g., character, floats, integers
- Learned about f strings, how to print messages using indentation and new lines

##Chapter 03--Lists
- Concept in of itself was not that challenging. 
- Know how to create a list, access certain positions, and then edit them. 
- Some methods to remember
  - Adding to list: append(), insert()
  - Removing: remove() pop() del
  - Sorting: sorted(), sort(), reverse()
  - How long is the list: len()
